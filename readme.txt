Content of source files:
- index.html - the HTML code for the first page of the application
- search.html - the HTML code for the second page of the application
- style.css - CSS code for styling both pages
- script1.js - JavaScript code for the first page
- script2.js - JavaScript code for the second page
- sw.js - JavaScript code for the Service Worker
- manifest.json - code for the web manifest

The service worker and web manifest are needed for the Progressive Web Application functionality.

To open the web application, extract all files from folder and open using the "index.html" file.
